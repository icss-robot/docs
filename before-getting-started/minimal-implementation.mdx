1. Search by a `query` string and a category to identify the
[Claimed Product (CP)](/before-getting-started/valuechecker-service-concepts#claimed-product-cp).

> \{api_url_path\}/query=iphone 11 64&client_category=Mobile%20Phones
```json excerpt
{
    "product_name": "Apple iPhone 11 64GB",
    "pid": 1032089319,
}
```

2. Search using a `pid` to return [Replacement Product (RP)](/before-getting-started/valuechecker-service-concepts#replacement-product-rp)
suggestions and prices.
<Tip>
    For locale pricing purposes, country is also a required variable.
</Tip>

> \{api_url_path\}/prices?pid=1032089319&country=nl
